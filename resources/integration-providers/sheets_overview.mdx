## Required Settings

* **Credentials required**
  - Client ID
  - Client Secret
  - Scopes

<Info>Learn how to create an OAuth app in Google Sheets [here](https://docs.gocobalt.io/resources/integration-providers/google_sheets).</Info>

* **Mandatory Scopes** 
1. https://www.googleapis.com/auth/userinfo.email
2. https://www.googleapis.com/auth/spreadsheets
3. https://www.googleapis.com/auth/drive.readonly
