## Required Settings

* **Credentials required**
    - Client ID
    - Client Secret
    - Scopes

<Info>Learn how to create an OAuth app in Slack [here](https://docs.gocobalt.io/resources/integration-providers/slack).</Info>

* **Mandatory Scopes** 
1. users:read(BOT)
2. users:read(USER)
