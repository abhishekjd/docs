<Note>
Trello is an **API Key** based integration and does not require the setup of an app.
</Note>

### Required Credentials of a Linked Account

To successfully connect with the **Trello** integration, a Linked Account or an end-user will need to provide the following to connect:

1. API Key
2. API Token

<Info>
To understand how a Linked Account can get the above mentioned credentials, refer [here](https://docs.gocobalt.io/resources/integration-providers/trello#getting-credentials-of-marketo).
</Info>


### Getting Credentials of Trello

To acquire the required credentials and connect a Linked Account, please follow the steps mentioned below:
<img height="200" src="/images/Trello/trello_config.png" alt="Connect a Linked Account"/>

1. Log in to your Trello account and go to [Power-Ups Admin](https://trello.com/power-ups/admin) page.
2. Click on the `Power-Ups` under **Apps** in the side menu and click on `New`.
<img height="200" src="/images/Trello/trello_navigation.png" alt="Navigation in Trello"/>

3. Enter the `Name`, select `Workspace`, fill all the required fields and click on `Create`.
4. Click on `Generate a new API key` > click on `Generate API Key` and copy the `API Key` and `Secret`(**API Token**).
<img height="200" src="/images/Trello/trello_credentials.png" alt="Getting Credentials"/>

The Linked Account or end-user now have all the credentials required to connect with Trello.

In Cobalt, you can create orchestrations of your use-cases using Trello actions and triggers. Following are the set of Trello actions and triggers
supported by Cobalt.

<Tabs>
  <Tab title="Actions">
     1. [HTTP Request](https://docs.gocobalt.io/resources/integration-providers/trello#actions-and-triggers) - Make HTTP API calls to any Trello documented REST APIs.
     2. [Create Card](https://docs.gocobalt.io/resources/integration-providers/trello#actions-and-triggers) - Create a new card in Trello.
     3. [Update Card](https://docs.gocobalt.io/resources/integration-providers/trello#actions-and-triggers) - Update a card in Trello.
     4. [Delete Card](https://docs.gocobalt.io/resources/integration-providers/trello#actions-and-triggers) - Delete a card in Trello.
     5. [Create Board](https://docs.gocobalt.io/resources/integration-providers/trello#actions-and-triggers) - Create a board in Trello.
     6.[Create List](https://docs.gocobalt.io/resources/integration-providers/trello#actions-and-triggers) - Create a new list in Trello.
  </Tab>

</Tabs>
