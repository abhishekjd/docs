---
title: 'Generate'
api: post /api/v2/public/session-token
---

This API provided a session token to authenticate a linked account. 

## Request description
#### Body Parameters
<ParamField body="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>

## Response description

<Tabs>
  <Tab title="200">
    <ResponseField name="token" type="string">
      Session Token. Expires in 24 hrs.
    </ResponseField>
  </Tab>
  <Tab title="500">
    <ResponseField name="status_code" type="integer">
    System generated status Code
    </ResponseField>
    <ResponseField name="error_type" type="string">
    System generated error type
    </ResponseField>
    <ResponseField name="message" type="string">
    Error message
    </ResponseField>
    <ResponseField name="request_id" type="string">
    System generated request id
    </ResponseField>
  </Tab>
</Tabs>
