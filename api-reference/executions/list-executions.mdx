---
title: 'Get all executions'
api: get /api/v2/public/execution
---

This API retrieves a list of all executions. 

## Request description
#### Headers
<ParamField header="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>
