---
title: 'Get execution'
api: get /api/v2/public/execution/:execution_id
---

This API fetches an execution's details. 

## Request description
#### Headers
<ParamField header="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>

#### Path Parameters
<ParamField path="execution_id" type="string" required>
</ParamField>
