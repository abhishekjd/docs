---
title: 'Delete linked account'
api: delete /api/v2/public/linked-account/:linked_account_id
---

This API lets you delete an existing linked account

## Request description
#### Path Parameters
<ParamField path="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>

## Response description
