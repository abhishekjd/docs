---
title: 'Get linked account by ID'
api: get /api/v2/public/linked-account/:linked_account_id
---

This API retrieves information about a specific linked account by using its unique identifier.

## Request description
#### Path Parameters
<ParamField path="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>
