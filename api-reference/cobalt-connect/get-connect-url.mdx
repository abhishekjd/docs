---
title: 'Generate Connect URL'
api: post /api/v2/public/connect-url
---

Returns the connection URL in format: `https://connect.gocobalt.io/{TOKEN}` 

## Request description
#### Headers
<ParamField header="Authorization" type="string">
Unlike other API's in this documentation. This api requires the session token as a bearer token. Please ommit the `x-api-key` from the header of this api.
  ```bash Header for Generate Connect URL api
  Authorization:  Bearer {SESSION TOKEN}
  ```
</ParamField>
#### Body Parameters
<ParamField body="color" type="string">
  Text Colour Hex
</ParamField>

<ParamField body="bgColor" type="string">
  Background Colour Hex
</ParamField>

<ParamField body="name" type="string">
  Customer name
</ParamField>
