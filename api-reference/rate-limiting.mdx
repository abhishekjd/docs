## Authentication
Cobalt apis uses api key to authenticate requests. You may use either of test or production keys depending upon the environment that you want to use.
You can find the api keys [here](https://app.gocobalt.io/your-app)

```bash
'x-api-key': '<Cobalt API key>'
```

#### Provide API keys as a header parameter
Provide your API key in the request headers as:

```bash Example Request
curl --location --request POST 'https://api.gocobalt.io/api/user' \
--header 'Content-Type: application/json' \
--header 'x-api-key:  <Cobalt API Key>' \
--data-raw '{
    "Example_Keyname": ""
}'
```

#### Supported Formats
Cobalt APIs support sending request body with the application/json content-type. All replies are also encoded in application/json; charset=utf-8

## Rate Limiting
Please be aware that Cobalt enforces rate limiting to ensure fair usage and prevent abuse. The rate limit is set to 10 requests per minute for each 
linked account. Make sure to manage your API usage accordingly to avoid throttling of subsequent requests.
