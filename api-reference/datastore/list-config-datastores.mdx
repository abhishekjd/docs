---
title: 'List Config Datastores'
api: get /api/v2/public/datastore/config/:slug
---

This API fetches list of all config datastores.

## Request description
#### Headers 
<ParamField header="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>

#### Path Parameters
<ParamField path="slug" type="string" required>
  Application Slug. Eg: `hubspot`
</ParamField>


## Response description

<Tabs>
  <Tab title="200">
    
    An array object containing names of all config datastores for the speicified application slug.

  </Tab>
  <Tab title="Error response">
    <ResponseField name="status_code" type="integer">
    System generated status Code
    </ResponseField>
    <ResponseField name="http_error_type" type="string">
    System generated error type
    </ResponseField>
    <ResponseField name="error_code" type="string">
    Specific error code to identify the error
    </ResponseField>
    <ResponseField name="message" type="string">
    Error message
    </ResponseField>
  </Tab>
</Tabs>
