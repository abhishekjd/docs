---
title: 'Get all events events'
api: get /api/v2/public/application/event
---

This API lists Cobalt [events](https://app.gocobalt.io/your-app#:~:text=Add%20events%20to%20send%20data%20from%20your%20app%20to%20other%20apps). 

## Request description
#### Path Parameters
<ParamField path="event_id" type="string" required>
  Event Id
</ParamField>

## Response description

<Tabs>
  <Tab title="200">
    <ResponseField name="_id" type="string">
      Event Id
    </ResponseField>
    <ResponseField name="event_url" type="string">
      Event URL for triggering the event
    </ResponseField>
    <ResponseField name="event_name" type="string">
      Name of the event
    </ResponseField>
    <ResponseField name="event_model" type="object">
      Event model for Trigger payload reference
    </ResponseField>
  </Tab>
  <Tab title="500">
    <ResponseField name="status_code" type="integer">
    System generated status Code
    </ResponseField>
    <ResponseField name="error_type" type="string">
    System generated error type
    </ResponseField>
    <ResponseField name="message" type="string">
    Error message
    </ResponseField>
    <ResponseField name="request_id" type="string">
    System generated request id
    </ResponseField>
  </Tab>
</Tabs>
