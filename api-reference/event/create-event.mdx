---
title: 'Create event'
api: post /api/v2/public/event
---

This api creates a Cobalt [event](https://app.gocobalt.io/your-app#:~:text=Add%20events%20to%20send%20data%20from%20your%20app%20to%20other%20apps). 

## Request description
#### Body Parameters
<ParamField body="name" type="string" required>
  Event Name
</ParamField>

<ParamField body="model" type="object">
  Event model. This model's properties are accessed in the workflow as trigger variables.
  ```JSON Example payload
    {
      "string_field": "string",
      "number_field": 12,
      "array_field": ["1","2"],
      "boolean_field": false
    }
  ```
  <Expandable title="example properties" defaultOpen="true">
      <ResponseField name="string_field" type="string">
          Example string
      </ResponseField>
      <ResponseField name="number_field" type="number">
          Example number
      </ResponseField>
      <ResponseField name="array_field" type="array">
          Example number
      </ResponseField>
      <ResponseField name="boolean_field" type="boolean">
          Example number
      </ResponseField>
  </Expandable>
</ParamField>

## Response description

<Tabs>
  <Tab title="200">
    <ResponseField name="_id" type="string">
      Event Id
    </ResponseField>
    <ResponseField name="event_url" type="string">
      Event URL for triggering the event
    </ResponseField>
    <ResponseField name="event_name" type="string">
      Name of the event
    </ResponseField>
    <ResponseField name="event_model" type="object">
      Event model for Trigger payload reference
    </ResponseField>
  </Tab>
  <Tab title="500">
    <ResponseField name="status_code" type="integer">
    System generated status Code
    </ResponseField>
    <ResponseField name="error_type" type="string">
    System generated error type
    </ResponseField>
    <ResponseField name="message" type="string">
    Error message
    </ResponseField>
    <ResponseField name="request_id" type="string">
    System generated request id
    </ResponseField>
  </Tab>
</Tabs>
