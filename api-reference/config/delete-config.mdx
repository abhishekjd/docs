---
title: 'Delete Config'
api: delete /api/v2/public/slug/:slug/config/:config_id
---

This API removes the specified config.

## Request description
#### Headers 
<ParamField header="linked_account_id" type="string" required>
  Unique customer identification id. For Eg: markwood@some_email.com
</ParamField>

#### Path Parameters
<ParamField path="slug" type="string" required>
  Application Slug. Eg: `docusign`
</ParamField>

<ParamField path="config_id" type="string">
 Id of the config to be fetched. If not provided, `linked_account_id` is considered by default.
</ParamField>
