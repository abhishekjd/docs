---
title: 'Create Webhook'
api: post /api/v2/public/webhook
---

This API registers a Cobalt [webhook](https://app.gocobalt.io/webhooks).

## Request description
#### Body Parameters
<ParamField body="webhook_url" type="string" required>
  Provide a url where you wish to listen to the event.
</ParamField>

<ParamField body="webhook_events" type="array" required>
  Array of the webhooks events to subscribe. Accepted values are, `Connection Created`, `Connection Deleted`, `Workflow Completed`, `Workflow Errored`, `Connection Expired`.
</ParamField>
<ParamField body="is_enabled" type="boolean">
  Determines whether the webhook is active or not.
</ParamField>

#### Query Parameters
<ParamField query="page" type="integer" default="1">
  Page number eg: 5
</ParamField>

<ParamField query="limit" type="integer" default="10">
 Records per page eg: 10
</ParamField>
