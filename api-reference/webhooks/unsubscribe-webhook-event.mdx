---
title: '(Deprecated) Unsubscribe from an event'
api: post /api/v2/public/webhook/unsubscribe
---

<Warning>
This API is now deprecated.
</Warning>

This API unsubscribes user from a Cobalt [webhook](https://app.gocobalt.io/webhooks) events.

## Request description
#### Body Parameters
<ParamField body="webhook_event" type="string" required>
  Webhooks event to unsubscribe. Accepted values are, `connection-created`, `connection-deleted`, `workflow-completed`, `workflow-errored`.
</ParamField>

## Response description

<Tabs>
  <Tab title="200">
    <ResponseField name="webhook_url" type="string">
      Registered webhook url
    </ResponseField>
    <ResponseField name="webhook_events" type="array">
      Subscribed events.
    </ResponseField>
  </Tab>
  <Tab title="Error">
    <ResponseField name="status_code" type="integer">
    System generated status Code
    </ResponseField>
    <ResponseField name="http_error_type" type="string">
    System generated error type
    </ResponseField>
    <ResponseField name="error_code" type="string">
    Specific error code to identify the error
    </ResponseField>
    <ResponseField name="message" type="string">
    Error message
    </ResponseField>
    <ResponseField name="request_id" type="string">
    System generated request id
    </ResponseField>
  </Tab>
</Tabs>
